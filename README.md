Task 12: Log Monitoring & Analysis

Overview

This task focuses on monitoring and analyzing authentication logs in Kali Linux to identify login activities and detect potential security threats.

Tools Used

Kali Linux

Terminal

Log File: /var/log/auth.log

Process

Accessed system authentication logs

Filtered successful and failed login attempts using grep

Reviewed logs for suspicious activity

Result

No failed login attempts detected

Legitimate user authentication confirmed

Conclusion

Log monitoring is essential for detecting unauthorized access and maintaining system security through continuous analysis of system logs
